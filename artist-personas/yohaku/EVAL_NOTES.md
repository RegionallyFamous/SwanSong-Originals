# Yohaku evaluation notes

## Discovery batch

Ten full-screen game types were generated with the same master rules and two
game-specific accent inks. The shared look remained stable across dark, light,
grid, side-view, instrument, and pseudo-3D compositions.

## Persistent strengths

- Game state reads before illustration detail.
- Cream/black value structure survives aggressive four-color reduction.
- Selection brackets, sockets, rings, pips, and silhouette icons transfer well
  between unrelated genres.
- The printed-zine voice remains visible without requiring decorative prose.

## Drift and response

- Large vehicle and landscape scenes accumulated excessive micro-detail.
  Response: native assets are hand-authored from the master grammar, not direct
  miniature crops.
- Generated boards sometimes differed from verified rules. Response: every
  renderer rebuilds geometry from the C game state.
- Some HUDs favored circular containers. Response: future prompts name the
  physical metaphor for each meter and cap repeated container shapes.

## Revisions

- v0.1: Orbital Courier established no-text, dominant-playfield direction.
- v0.5: nine-game discovery added explicit 82–88 percent playfield and complete
  icon-state inventory rules.
- v1.0: added geometry authority, native-detail limits, renderer regression
  tests, composition adapters, and per-game learning records.

## Reference candidates

Orbital Courier, Mote Sound Terminal, Radio Ghost, Turncoat Tactics, Pocket
Kaiju Observatory, Rotate Dungeon, and Bug Witch are clean 5/5 exemplars.
Scrapframe Garage, Harpoon Moon, and One Last Lap are useful 4/5 stress tests
whose native conversions intentionally simplify source detail.
