# Yohaku Prompt Recipes

## Formula

```text
Requested subject + requested subject identity lock + persona style invariants + composition adapter + motif policy + hard avoid list.
```

## Discovery Prompt Template

```text
Create [subject/action] as an original visual style exploration for Yohaku.
Preserve the identity lock: [identity cues].
Style hypothesis: [shape, line, rendering, palette, texture, composition].
Avoid: [hard avoid list].
No text, pseudo-text, logos, signatures, watermarks, or named artist imitation.
```

## Benchmark Prompt Template

```text
Draw [benchmark subject] in the Yohaku original house style.
Preserve these style invariants: [style invariants].
Preserve these subject requirements: [identity or object requirements].
Use this composition: [composition].
Avoid: [negative prompt].
```
