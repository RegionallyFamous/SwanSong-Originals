# Originality and public-release policy

SwanSong Originals is a public collection of newly authored microgames. Its
game names, characters, settings, code, sound sequences, symbols, interfaces,
ASCII art, and concept artwork are original to this project.

Mote Sound Terminal is an original spacecraft music utility featuring an
angular, crystal-shaped floating maintenance companion. It uses three original
procedural synth sequences and does not contain copied melodies, recordings,
voices, characters, emblems, vehicles, interfaces, or extracted assets.

Platform and toolchain names appear only where needed to describe technical
compatibility. They do not imply affiliation, sponsorship, or endorsement.

Before publishing a release:

1. Search source, documentation, and generated-art prompts for third-party
   franchise names or copied terminology.
2. Verify that all ten default ROMs build from the public tree.
3. Run the structural, gameplay-path, and SwanSong-frame test lanes.
4. Review new art and audio against their recorded source/provenance notes.
