# SwanSong Originals 3.3.0 — Proven on SDK 0.5

This release advances all ten cartridges to the content-addressed SwanSong SDK
0.5.0 runtime while preserving their approved presentation, controls, outcomes,
and deterministic reset behavior.

## Highlights

- Every manifest pins SDK 0.5.0 and payload revision
  `sha256:905d1b7683ea55aebb90703bc4dc708ae7a436c98dae1474e67c9df89601a35c`.
- The anthology now exercises the SDK's expanded gameplay runtime, corrected
  storage mapping, isolated normal/trace build variants, and exact package
  identity checks.
- All ten production ROMs build at 128 KiB and pass their portable C gameplay
  contracts, cartridge validation, art checks, and manifest checks.
- All 61 declared fresh-boot scenarios complete in SwanSong, including
  interaction, success, failure, reset, distinct One Last Lap outcomes, and
  bit-exact deterministic replays.

## Validation

Release acceptance uses SwanSong exclusively. The release was checked through
Wonderful builds, production C host tests, resource validation, and fresh-boot
SwanSong execution with PNG and WAV evidence. No Mednafen or alternate emulator
participates in the build or release gate.
