# SwanSong Originals 3.2.0 — Complete Outcomes

This release moves every cartridge to SwanSong SDK 0.4.0 and turns the first
ten games into a stricter regression suite for future homebrew work.

## Highlights

- Normal frame presentation no longer runs a whole-map and whole-sprite
  profiler sweep. Games keep their input/update cadence while exact resource
  diagnostics remain available on demand.
- Every success plan reaches the documented ending or utility outcome instead
  of stopping at a first pickup, lock, tag, room, photo, lap, or legal move.
- Mote Sound Terminal now plays continuous four-voice patterns; pause/resume
  preserves phase and the scope responds to live channel activity.
- Orbital Courier caches its viewport and HUD, redrawing the full world only
  when the camera changes.
- One Last Lap proves both cooperative and solo endings through distinct
  SwanSong scenarios.
- Game-specific feedback, control contracts, and boundary visuals were aligned
  with the production models and expanded host tests.
- SwanSong Playtester 0.3.1 now rejects milestone-only success evidence and
  separates input delivery from missed game-update frames during diagnosis.

## Validation

Release acceptance uses SwanSong exclusively: production C host tests,
Wonderful builds and resource gates, fresh-boot interaction/success/failure/
reset scenarios, inspected PNG/WAV evidence, and unchanged deterministic
replays. No Mednafen or alternate emulator participates in the build or release
gate.
